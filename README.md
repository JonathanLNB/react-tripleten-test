Most of the mistakes in the project are described in the checklist, but there are also errors
which do not directly follow from it because it is difficult to formalize the whole range of errors,
which students can make. Such errors should also be noted.

The project is a small one-page site where the user can upload their photos, like and delete them, and edit the user info.

The following technologies were used in the project:
* adaptive layout through media queries
* flexbox layout
* grid layout;
* Nested BEM;
* React + Hooks
* React Context
* React Router
* JWT

npm i - install dependencies
npm run build - build the project
npm run dev - run the project
